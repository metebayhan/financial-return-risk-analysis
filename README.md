# Financial Return & Risk Analysis

Python project for analyzing asset performance and risk.

## Features

- Log return calculation from historical prices  
- Annualized returns using log aggregation  
- Variance and standard deviation for volatility measurement  
- Pearson correlation for asset relationship analysis  
- Modular structure with utils.py  

## Tech

- Python  
- Math library  

## Structure

- main.py → runs analysis  
- utils.py → reusable financial functions  

## Notes

Built as part of personal learning in financial data analysis.
